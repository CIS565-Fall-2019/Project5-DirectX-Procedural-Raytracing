## Introduction

![](images/dxr.gif)

This project implemented ray tracing for procedural geometry using Direct X 12 Raytracing API.

Keyboard Shortcut:

 - `C` - enable/disable camera animation.
 - `G` - enable/disable geometry animation.
 - `L` - enable/disable light animation.
